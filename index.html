Vilka är vi.

DiagTech-Webbutveckling är ett ny startad företag av personer som brinner för att skapa nya lösningar som hjälper individer eller företag att lösa deras problem på det digitala planen. Vi älskar att skapa en User vänlig Design, samt funktonella Appar eller Hemsidor.

Våra medarbetare på DiagTech kan hjäpla dig med ovanstående tjänster.

På design och UX avdelningen använder vi oss av Figma och Ilustrator

På Hosting avdelningen hjälper vi dig med webbhotel förvaltning och service.

På CEO avdelningen optimerar vi din sida så att den hamnar högst i sökningar

Vi använder nädanstående verktyg för att bygga hemsida eller webb app
